HTML

HTML - это язык гипертекстовой разметки, при помощи которого создаётся основная структура сайта, например текстовое содержимое, картинки, ссылки, поля ввода данных и т.д.

CSS

CSS - это каскадные таблицы стилей, описывающие внешний вид web-страницы, например цвет фона и текста, расположение отдельных элементов, анимации и т.д.

Работа с графикой

При создании сайта часто приходится использовать различные картинки и анимации, поэтому очень важно разбираться в технологиях построения графических элементов.

JavaScript

JavaScript - это интерпритируемый язык программирования, который получил наиболее широкое применение в браузерах, как язык сценариев для придания интерактивности веб-страницам
